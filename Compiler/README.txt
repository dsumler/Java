In order to run the compiler, navigate to the "src" folder in your command prompt
and type in "javac Parser.java" followed by "java Parser" to run the program.
You will be prompted to enter a file name. There are some sample programs in 
the "src" folder, but in order to add your own, they must be placed in the "src" folder.

For a list of Bugs and Limitations, please look at the documentation.
