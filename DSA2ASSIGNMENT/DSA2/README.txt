In order to run the program, navigate to the src folder in your command prompt.

If you want to run Part 1 - Boolean Satisfiability, run the following commands:
javac Part1.java
java Part1

If you want to run Part 2 - Data Compression, run the following commands:
javac Part2.java
java Part2

A known bug in Part1.java is that infinite loops may be present in the program, depending on
what expression is entered. Please check the documentation for more detailed information.

Any text files that you wish to be read by the program "Part2.java" must be placed in the src
folder.
